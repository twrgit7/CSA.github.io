# Cyber Sith Academy (TCSA)

Welcome to the Cyber Sith Academy (TCSA), a platform dedicated to mastering cybersecurity with a dark-side twist. Train your skills, solve vulnerable machines, and level up your practical offensive and defensive security knowledge.

Learn and practice cybersecurity by playing with our VMs:

## About

The Cyber Sith Academy is an innovative cybersecurity learning platform that combines the principles of the Sith with cutting-edge cybersecurity techniques. Our mission is to empower individuals to embrace the darkness within the digital realm and become formidable guardians of the cyber galaxy.

## Features

- Dive into practical cybersecurity labs.
- Learn from experienced community creators.
- Practice ethical hacking and digital defense techniques.
- Build skills in enumeration, exploitation, privilege escalation, and hardening.
- Access a growing repository of VMs and writeups.

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. In **Settings → Pages**, set **Source** to **GitHub Actions**.
3. Push to the `main` or `master` branch (or run the workflow manually from the Actions tab).
4. Wait for the **Deploy Jekyll site to Pages** workflow to finish.

No custom domain is required.

Your site URL will be one of these:
- `https://<your-username>.github.io/` (if this repo is named `<your-username>.github.io`)
- `https://<your-username>.github.io/<repo-name>/` (for a regular project repo)

If you later buy a domain, then add a `CNAME` file and configure it in **Settings → Pages**.
